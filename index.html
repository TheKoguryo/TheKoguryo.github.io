오라클 클라우드 초보자를 위한 기술 문서 모음 사이트

무엇인가 익숙하지 않고 낯선 것을 처음 시작하는 것은 참 쉽지 않습니다. 오라클 클라우드를 처음 사용하는 사용자 또한 익숙하지 않고 낯설음으로 인해 어디서 시작해야 할지? 어떻게 해야 할지 어려움이 부딪히게 될 것입니다. 본 사이트는 오라클 클라우드를 처음 사용하는 사람들에게 도움이 되고자 만들어졌습니다. Oracle Cloud Infrastructure를 이해하고 기본적인 Compute VM을 만들어서 사용하는 과정을 통해 Compute 서비스는 물론 다른 서비스들을 사용하는 데 도움이 되었으면 합니다.

처음은 Compute VM을 위주로 시작하겠지만, 점차 다른 서비스로 다루는 대상을 늘려가고자 합니다.
