Oracle Cloud Native

Oracle Cloud Native는 최근 각광받고 있는 쿠버네티스, 컨테이너, 서버리스 함수 등을 포함하여 여러가지 서비스와 소프트웨어 등을 Oracle Cloud Infrastructure(OCI) 기반으로 제공하고 있습니다. 컨테이너 기반 기술을 사용하는 데 있어서 개발자들은 운영 시간을 줄이고, 빠르게 컨테이너 환경을 구축하여 사용할 수 있습니다. 본 사이트에서는 Oracle Container Enginer for Kubernetes 중심으로 OCI 제공하고 있는 컨테이너 관련 서비스들을 설명하고자 합니다.
