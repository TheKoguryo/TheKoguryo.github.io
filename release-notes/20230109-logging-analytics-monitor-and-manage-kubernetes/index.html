Logging Analytics: Monitor and manage Kubernetes with our Marketplace app

  • Release Notes - Logging Analytics: Monitor and manage Kubernetes with our Marketplace app

업데이트 사항

GitHub Source를 통해 OKE 클러스터에서 Fluentd를 통해 로그를 수집하여 OCI Logging Analytics로 모니터링하는 툴을 제공하고 있습니다. 이번 업데이트는 이를 Kubernetes Monitoring and Management 란 이름으로 OCI 마켓플레이스에서 제공하게 되었다는 업데이트 입니다. 마켓플레이스로 가면, Resource Manager를 통해 대상 OKE 클러스터에 수집용 모듈을 설치하고, OCI Logging Analytics로 수집되고, 또한 Kubernetes 모니터링용 대쉬보드 추가 제공합니다.

자세한 사항은 다음을 참고합니다.

  • 5.2.3 Logging Analytics를 사용한 모니터링
