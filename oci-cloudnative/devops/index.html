DevOps 서비스

  • OCI DevOps 서비스는 OCI에서 제공하는 관리형 CI/CD 서비스입니다. 애플리케이션 소스 코드 저장소부터 실제 배포까지를 위한 툴을 제공합니다.

    • 공식 소개 문서

      https://docs.oracle.com/en-us/iaas/Content/devops/using/devops_overview.htm

    OCI DevOps 주요 구성 요소

    • DevOps 프로젝트

      CI/CD 워크플로우를 구성하는 자원들의 논리적인 그룹으로 단일 프로젝트 내에 소스 코드 저장소, 빌드, 배포 파이프 라인 및 추가 관리 자원들을 포함합니다.

    • Code Repository

      소스 코드 저장소인 Git Repository 입니다.

    • Build Pipeline

      CI(Continous Integration)을 위한 소스 코드 빌드를 위한 파이프 라인을 구성하는 기능을 제공합니다. OKE 기반에서는 소스 코드를 빌드하여 컨테이너 레지스트리인 OCIR에 컨테이너 이미지를 저장하는 과정을 구성하는 영역입니다.

    • Deployment Pipeline

      CD(Continous Deploy)를 위한 배포 파이프 라이을 구성하는 기능을 제공합니다. OKE 기반에서는 OCIR에 새로이 저장된 컨테이너 이미지를 배포 YAML Artifact를 통해 OKE에 배포하는 과정을 구성하는 영역입니다.

    • Artifact

      빌드 산출물(예, 컨테이너 이미지), 배포시 사용할 자원(YAML 설정 파일) 등 소스 코드 이외에 빌드/배포 파이프라인에서 사용될 자원을 저장하는 저장소입니다.

    • 그외 외부 저장소 연결, 로그, 트리거 등 연관 기능을 제공합니다.
