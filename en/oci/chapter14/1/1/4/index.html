14.1.1.4 API Public Key Registration

  1. Go to the user detail page registered in OCI Config File

  2. Select API Keys from Resources at the bottom left

  3. Click Add API Key

  4. Check the generated API Public Key after running oci setup config

    [opc@bastion-host ~]$ cat ~/.oci/oci_api_key_public.pem
-----BEGIN PUBLIC KEY-----
MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8AMIIBCgKCAQEAycclV86OzQ+x6I13HEbe
...
gCO1GdLyJNS08Zv1uiW6j1IYIszrmr6XK482Vf3r48u8ZkFgBBlsPjU03R9H1x52
dwIDAQAB
-----END PUBLIC KEY-----

  5. Add API Public Key to the generated API Public Key

  6. API Public Key registration for OCI CLI completed
