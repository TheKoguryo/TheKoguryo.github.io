Oracle Cloud Infrastructure

OCI is an abbreviation of Oracle Cloud Infrastructure, and refers to the basic infrastructure environment required to implement cloud services in the customer’s data center, that is, the Oracle Cloud infrastructure including computer servers, networks, storage, security, and resource management.

Based on this Oracle Cloud Infrastructure, we are providing other high-level services of platforms (Oracle Cloud Platform) and applications (Oracle Cloud Software). The purpose of this document is to help you understand and get started with the concepts more easily, even if you are not familiar with Oracle Cloud Infrastructure, which is the basis of all Oracle cloud services for the first time.
