Technical Documents Site for Oracle Cloud Beginners

It is not easy to start something unfamiliar for the first time. Users who are new to Oracle Cloud are also unfamiliar, so where do they start? You will run into difficulties figuring out what to do. This site is intended to be helpful to those new to Oracle Cloud. We hope that the process of understanding Oracle Cloud Infrastructure and creating and using a basic Compute VM will help you to use the Compute service as well as other services.

We will start with Compute VMs at first, but we want to gradually increase the number of targets covered by other services.
