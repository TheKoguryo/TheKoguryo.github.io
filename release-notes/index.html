컨테이너관련 OCI 주요 기능 업데이트

2023-10-17, Cache with Redis is now available cache redis

이제 Cache with Redis를 통해 관리형 Redis 서비스를 제공합니다.

2023-10-04, Support for Kubernetes volume clones oke volume clone

이제 CSI volume plugin은 Kubernetes 볼륨 복제를 지원합니다. 내부적으로는 OCI Block Volume 서비스의 복제 기능을 활용합니다.

2023-06-20, Support for Kubernetes volume snapshots oke volume snapshot

CSI volume plugin에서 쿠버네티스 볼륨 스냅샷을 지원합니다. Block Volume을 사용하여 만든 Persistent Volume을 쿠버네티스 볼륨 스냅샷을 요청하면, OCI Block Volume Backup을 통해 스냅샷이 생성됩니다. 재해복구 관점에서 볼륨 스냅샷을 만들고, 복구시 사용할 수 있는 기능을 제공합니다.

2023-05-09, Support to store image pull secrets in OCI Vault container instances vault

마켓플레이스를 통해 Kubernetes Monitoring and Management 구성 툴을 제공합니다. Resource Manager를 통해 설치하면 OKE에서 수집된 로그, 메트릭을 OCI Log Analytics에서 모니터링하는 기능을 제공합니다.

2023-04-26, Support for service mesh and performance improvements when using the VCN-Native CNI plugin oke VCN-Native Pod Networking CNI

OCI VCN-Native Pod Networking CNI을 사용하는 경우, Pod가 VCN의 IP의 할당받아, 해당 IP로 직접 통신할 수 있습니다.

2023-01-09, Logging Analytics: Monitor and manage Kubernetes with our Marketplace app logging analytics kubernetes marketplace

마켓플레이스를 통해 Kubernetes Monitoring and Management 구성 툴을 제공합니다. Resource Manager를 통해 설치하면 OKE에서 수집된 로그, 메트릭을 OCI Log Analytics에서 모니터링하는 기능을 제공합니다.

2022-12-14, Support for dynamically provisioning PVCs in File Storage service dynamic provisioning pvc file storage service kubernetes

File Storage 서비스를 OKE에서 Persistent Volume으로 사용하는 경우 이제 Dynamic Provisioning을 지원합니다.

2022-12-06, Container Instances is now available container-instances container

컨테이너를 간편하게 실할하여 사용할수 있는 Container Instances를 서비스로 제공합니다.

2022-08-02, Code Editor is now available cloudshell code editor

오라클 클라우드 콘솔에서 코드 에디터를 제공합니다. Cloud Shell과 동일한 사용자 환경을 사용합니다.

2022-08-02, Cloud Shell now offers Private Access cloudshell private access

Cloud Shell에서 Private IP를 가지는 자원에 대한 접근을 지원합니다.

2022-07-26, CloudShell now offers GraalVM Enterprise JDK 17 and Native Image cloudshell graalvm

CloudShell에서 GraalVM Enterprise JDK 17을 제공합니다.

2022-07-12, Support for VCN-native pod networking oke vcn-native pod

OCI VCN-Native Pod Networking CNI을 사용하는 경우, Pod가 VCN의 IP의 할당받아, 해당 IP로 직접 통신할 수 있습니다.

2022-06-28, Support for worker node deletion, along with new cordon and drain options oke worker node deletion

특정 Worker Node를 삭제하는 기능을 제공합니다.

2022-06-08, Private endpoints for Resource Manager resource manager terraform

Resource Manager에서 Private Endpoint를 통해 인터넷을 거치지 않고, Private 환경에서 있는 자원에 대한 접근을 지원합니다.

2022-06-01, Support for OKE images as worker node base images oke oke image

Worker Node의 베이스 이미지로 범용 이미지가 아닌 OKE 전용 이미지를 사용하므로 이전보다 더 빠르게 클러스터 생성 및 확장이 가능합니다.

2022-06-01, Support for fault domains in node pool placement configuration oke node pool fault domain

Node Pool 생성시 Availability Domain을 설정시 Fault Domain을 사용자가 지정할 수 있습니다.

2022-05-26, Support for CSI metrics metric csi

Persistent Volume에 대한 용량에 대한 모니터링 지원을 추가적인 메트릭을 지원합니다.

2022-05-18, Support for Kubernetes version 1.23.4 OCI Kubernetes Engine oke

쿠버네티스 지원 버전이 업데이트 되었습니다. 이제 1.23.4을 새롭게 지원합니다.

2022-05-17, Detect vulnerabilities from the DevOps build pipeline DevOps ci/cd

Application Dependency Management (ADM) 서비스와 연계하여 DevOps 빌드 파이프라인에서 라이브러리 패키지에 대한 취약점 검사기능을 제공합니다.

2022-05-17, New features for DevOps DevOps ci/cd

OCI DevOps 서비스가 Bitbucket Cloud, Helm Chart, 취약점 검사 기능을 제공합니다.

2022-05-10, OCI Search Service with OpenSearch is now available searchservice opensearch

OpenSearch 기반의 관리형 서비스로 OCI Search 서비스가 출시되었습니다.

2022-05-04, Accelerate function start-ups using provisioned concurrency Functions fn serverless

함수 실행을 위한 초기 응답지연을 최소화하기 위해 항상 provisioned concurrency 만큼 가용 환경을 지원합니다.

2022-04-11, Support for PVC block volume performance levels oke pvc vpu

Persistent Volume으로 사용하는 Block Volume에 대해서 성능 레벨을 지정할 수 있게 되었습니다.

2022-03-29, DevOps now supports deployment to private Kubernetes API endpoints DevOps ci/cd

DevOps 서비스가 프라이빗 쿠버네티스 API 엔드포인트를 지원합니다.

2022-03-29, DevOps adds Blue-Green and Canary deployment strategies DevOps ci/cd

DevOps 서비스에서 Blue/Green 배포 전략, Canary 배포 전략을 새롭게 지원합니다.

2022-03-21, Support for Kubernetes version 1.22.5 oke

쿠버네티스 지원 버전이 업데이트 되었습니다. 이제 1.22.5을 새롭게 지원합니다.

2022-03-15, Support for OCI Network Load Balancers oke

OKE에서 Service Type을 Load Balancer를 사용할때 이제는 OCI Network Load Balancer을 추가적으로 지원합니다.

2022-03-15, Container image scanning using the Console Vulnerability Scanning vss

Oracle Cloud 콘솔에서 Vulnerability Scanning 서비스를 통해 컨테이너 이미지 스캐닝하여 취약점을 분석하는 기능을 지원합니다.

2022-03-03, Support for function invocation logs Functions fn serverless

서비스 중인 Function을 클라이언트가 호출을 로그 정보를 확인하는 기능을 지원합니다.

2022-02-28, Support for PVC block volume expansion oke pvc

OKE에서 Block Volume을 Persistent Volume으로 사용하는 경우, 할당 받은 이후에도 사용 중인 볼륨의 크리를 확장하는 기능을 지원합니다.

2022-02-11, Support for 2 GB functions Functions fn serverless

Functions 서비스에서 하나의 함수에서 사용할 수 있는 최대 메모리를 2GB까지 지원합니다.
